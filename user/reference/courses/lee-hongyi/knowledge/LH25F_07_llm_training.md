## 核心教学内容提炼

### 1. 三阶段概览（面向导论课学生的版本）

[Fact] 李宏毅把 LLM 训练比喻为人的学习过程：

> "今天就是标准的三阶段学习。Pretrain = 让语言模型去熟悉人类的语言长什么样子。SFT/RLHF = 不只是熟悉人类的语言，它要成为一个**有用的 AI**，应该学习怎么样做正确的应对进退。"

### 2. Pretrain — 海量数据的文字接龙

[Fact] Pretrain 的目标简单粗暴：

> "Pretrain 就是让语言模型去看大量网路上的文字，学会文字接龙。它不需要人类标注答案——只要有纯文字就行。"

### 3. SFT 的成功建立在 Pretrain 之上

[Fact] 李宏毅用精彩的论证说明 SFT 离不开 Pretrain：

> "SFT 的成功其实是站在 Pretrain 这个巨人的肩膀上。没有先做 Pretrain，光是有 SFT 是没有办法成就一个好的语言模型的。"

> "如果只有 SFT 的话会怎样？因为人类没有办法真正准备非常大量的 SFT 资料——需要有人想问题、找人来写标准答案——这是一个非常大的工程。假设你准备的 SFT 资料很少，机器学习的时候就会非常容易 Overfitting。"

### 4. Distillation — 用强模型教弱模型

[Fact] 关于"自我学习"的讨段：

> "你可以把半句话当作问题，然后叫 ChatGPT 去做文字接龙接出后半句。你自己的模型就把比较好的语言模型接出来的结果当作老师。神奇的事情是——**这一招是有用的！**"

### 5. 一二三阶段的统一视角

[Fact] 最精彩的总结：

> "一二三阶段都是在学文字接龙，只是训练的资料不一样而已。RLHF 的训练方法看起来非常不一样，但实际上真正的 update 方式跟 Supervised Learning 并没有本质上的差别。唯一的差别只是现在**不只要拉近，有时候要拉远**。"
