## 核心教学内容提炼

### 1. 生成式 AI 的本质 = 文字接龙

[Fact] 李宏毅的核心定义：

> "生成式 AI 的基本原理就是**文字接龙**。给它一个未完成的句子，它预测下一个 Token。"

机器学习就是从数据中找函式 f：输入一段文字 → 输出下一个 Token。

### 2. Token ≠ 字

[Fact] 李宏毅特别强调：

> "Token 不等于字。中文一个字可能被切成多个 Token；英文一个词也可能被切成多个 Token。不同模型的 Tokenizer 不同，所以同样的句子在不同模型里的 Token 数不一样。"

### 3. 从文字接龙到"AI 能做的所有事"

[Fact] 李宏毅展示了几乎所有 AI 能力都可以归结为文字接龙：
- **翻译**：输入中文句子 → 接龙出英文翻译
- **摘要**：输入长文 → 接龙出摘要
- **代码**：输入需求 → 接龙出代码
- **推理**：输入问题 → 接龙出推理过程 + 答案

> "你可能觉得很不可思议，AIGC 做了那么多厉害的事情，底层居然就是文字接龙。但确实如此。"

### 4. 模型的能力来自训练，不是"被编程"

[Fact] 他反复强调一个被大众误解的观点：

> "模型并没有被『程式设计』成会做翻译或写代码。它只是看了大量数据，学会了文字接龙的规律。当你给它一段看起来像翻译任务的输入时，它会接龙出翻译结果。"

### 5. Hallucination（幻觉）的本质

[Fact] 李宏毅解释为什么模型会"胡说八道"：

> "模型就是在做文字接龙，它每次只是在预测最可能的下一个 Token。它没有『知道自己不知道什么』的能力。所以当你问它不知道的问题时，它还是会继续自信地接龙下去——这就是幻觉。"

### 6. 拒绝行为是后天训练出来的

[Fact] 关于模型的"安全性"：

> "模型本来什么都会回答。拒绝是后来通过 Alignment 训练出来的。研究发现，你可以去干预模型内部的 representation，找到『拒绝的方向』，然后把那个方向减掉，模型就又什么都回答了。"
